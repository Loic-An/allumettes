## Run Locally

Clone the project

```bash
  git clone https://github.com/Loic-An/allumettes
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the game

```bash
  python game.py
```
